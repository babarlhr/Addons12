Overtime Requests And Payroll Integration

This module add below features which can be used to manage overtime requests:

    * Overtime Requests: This will be request by employee.

    * Multiple Overtime Requests: This request can be created by HR Manager or Department Manager on behalf of multiple employees together.

    * Integrate Overtime Requests in Employee Payslips: This will show payslip lines in payslip of employee.

Note: This module add new group called "Department Manager(Overtime)" under Usability group.

Currently module will use fixed cost of employee on salary rule. Means you have to use fixed cost price / hour to calculate overtime allowance.



Menus:
  • Employees/Overtimes
  • Employees/Overtimes/Overtime Requests
  • Employees/Overtimes/Overtime Requests to Approve
  • Employees/Overtimes/Overtime Requests to Approve By Department
  • Employees/Overtimes/Multiple Overtime Requests
  • Employees/Overtimes/Multiple Requests to Approve
  • Employees/Overtimes/Multiple Requests to Approve By Department

Overtime Request Form

This form will allow employee to create overtime request and submit it to HR Manager and Department Manager for approval.

Multiple Overtime Request Form

This form will allow HR Manager or Department Manager to create overtime request on behalf of multiple employees using single request, multiple employees can be selected on form and approval flow will be as it is like single overtime request form. Once request will reach to Done state, it will automatically create overtime request for each employee and make it Done too so that we can track it on employee payslips.

Overtime Salary Rule

This salary rule will be added by module and that can be selected on salary structure of employee that will make computation of payslip and show that allowance on payslip lines.

Overtime Allowance in Employee Payslip

This will allow integration of employee payroll and overtime request. (Either from single overtime request or multiple overtime request.)
